# FFF1 简单回显规则（当前 Mac 默认）

对特征 **FFF1** 的任意写入，Mac 回复固定格式：

```
[0x00][0xAA] + 对方发来的原始字节
```

## 示例

| 手机写入 (hex) | Mac 回复 (hex) |
|----------------|----------------|
| `48 65 6C 6C 6F` (`Hello`) | `00 AA 48 65 6C 6C 6F` |
| `01 02 03` | `00 AA 01 02 03` |
| (空) | `00 AA` |

## 如何收到回复（自动，不必 Read）

BLE 规定：Peripheral **不能**在对方未订阅时主动塞数据，只能：

1. 手机对 FFF1 **打开 Notify / 订阅通知**（三方 App 里点铃铛或 Subscribe）
2. 再 **Write** 任意内容
3. Mac **自动 Notify 推送** `00 AA + 你写的内容` → 手机在「通知/Notify」里直接看到

若 **没开 Notify**，只能手机再 **Read FFF1** 才能读到（当前你看到的就是这种情况）。

无 JSON 协议、无周期推送。
